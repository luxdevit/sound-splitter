# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-04-15
### Fixed
- Fixed an issue where the `torchaudio` backend failed to export `.wav` files by adding the `soundfile` dependency.

## [1.0.0] - 2026-04-15
### Added
- Initial release of Sound Splitter Studio.
- Local first web interface for vocal separation via Demucs.
- Fast, secure GPU-accelerated local execution.
- 1-click startup scripts for macOS.
